**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 新闻信息
5. 商品信息
6. 个人中心
7. 收货地址
8. 商品收藏
9. 商品订单
10. 购物车
11. 修改密码
12. 商品信息管理
13. 商品收藏管理
14. 商品信息订单管理
15. 新闻信息管理

**商家功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 商品信息管理
6. 商品信息订单管理
7. 新闻信息管理



**管理员功能：**
